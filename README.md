# Template Repository

This is [career page file][readme] of Recursive Zero

## Pre-requisite

- Node 18+
- Typescript v 5.0+
- A Browser
- an IDE ( VS Code Preferred)

## How to start

- Navigate to the repo from terminal
- Install dependencies using below command

```sh
> npm install
```

then

```sh
> npm run start
```
